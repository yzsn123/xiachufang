### 下厨房发现

**参数** 无
**返回值** 
```json
{
    message: 'ok',
    code: 0,
    data: {
        picUrl,
        description,
        username,
        num(点赞数量)
    }
}
```
### 下厨房发现为你推荐的详情
**method:** get
**url:** /api/kitchen/discover/detail
**参数:** id
**返回值**  
```json
{
  message: 'ok',
  code: 0,
  data: {
      picUrl,
      collectNum(收藏数),
      browseNum(浏览数),
      headImg,
      userId,
      publishTime(发布时间),
      isAttentoin(是否被关注 true:关注),
      course(过程)
  }
}
```
